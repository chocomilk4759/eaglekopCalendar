# 연간 계획표 (Vercel 정적 배포, 라이트 테마 + 프리셋 아이콘)

- 라이트(흰색) 테마
- 우측 사이드바에 프리셋 아이콘 목록(클릭 시 메모에 삽입, 메모창 닫힌 경우 클립보드 복사)
- 월/년 선택 시 달력 자동 갱신
- 날짜별 메모 LocalStorage 저장
- 백업/복원(JSON)

## 배포(간단)
1. 이 폴더를 ZIP으로 업로드하거나 GitHub에 올려 Vercel에 연결
2. Framework Preset: Other
3. 정적 사이트로 즉시 배포
